## Features
- Automatically finds all JS files from target URL
- Extracts API endpoints from JS files
- Finds hidden paths, URLs, and parameters
- Saves results to output file automatically
- Color coded terminal output

## Installation

git clone https://github.com/ambuj9004-blip/js_endpoint_hunter.py
cd js_endpoint_hunter.py
pip install requests colorama

## Usage

python js_endpoint_hunter.py https://example.com

## What it Finds
- API endpoints (/api/v1/users)
- Hidden paths and files
- External URLs
- Fetch/Axios requests
- PHP, JSP, JSON endpoints
